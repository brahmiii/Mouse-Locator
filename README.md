# Mouse-Locator
This basic program will Give you mouse position Updatly.<br>
Using Pyautogui to locate mouse (x,y) ! <br>
```PIP INSTALL PYAUTOGUI```<br>
Using Kivy for the GUI ! <br>
```PIP INSTALL KIVY```<br>
